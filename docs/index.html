Add Medium for dual publishing Free
Publish same articles to Medium + dev.to — doubles reach, zero extra LLM cost.
Secret needed: MEDIUM_INTEGRATION_TOKEN
How to set up
  1. Sign into medium.com → Settings → Security and apps → Integration tokens
  2. Click Generate integration token, copy it
  3. In your GitHub repo → Settings → Secrets and variables → Actions → New repository secret
  4. Name: MEDIUM_INTEGRATION_TOKEN · Value: paste your token

Add OpenRouter for free-tier LLM fallback Free
OpenRouter exposes Gemini 2.0 Flash, Llama 3.3, Mistral 7B — all free. Reduces Groq quota pressure.
Secret needed: OPENROUTER_API_KEY
How to set up
  1. Go to openrouter.ai → sign up (free, no credit card needed for free-tier models)
  2. Dashboard → API Keys → Create Key, copy it
  3. GitHub repo → Settings → Secrets → Actions → New repository secret
  4. Name: OPENROUTER_API_KEY · Value: paste your key
Free models: google/gemini-2.0-flash-exp:free, meta-llama/llama-3.3-70b-instruct:free, mistralai/mistral-7b-instruct:free

Add Gemini API key for direct Google AI access Free tier
Gemini 2.0 Flash has a generous free tier (1 500 req/day). Adds a fast, capable LLM option before Groq.
Secret needed: GEMINI_API_KEY
How to set up
  1. Go to aistudio.google.com → sign in with a Google account
  2. Click Get API key → Create API key (select a Google Cloud project or create one)
  3. Copy the key
  4. GitHub repo → Settings → Secrets → Actions → New repository secret
  5. Name: GEMINI_API_KEY · Value: paste your key
Free tier: 1 500 requests/day on gemini-2.0-flash, 15 req/min. No billing required.
