# LLM Providers

For the no-ID/free path, prefer Groq or Gemini first, then OpenRouter free
models if you need more fallback capacity. Avoid Claude/Anthropic when the goal
is to stay away from premium or paid access.

## Supported Providers
* Gemini
* OpenRouter
* Groq
* Anthropic (paid fallback, avoid for no-ID/free setup)
